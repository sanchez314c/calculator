## [1.0.0] - 2024-09-13

### Added
- Initial release of Calculator desktop application
- Basic calculator functionality (addition, subtraction, multiplication, division)
- Modern dark-themed user interface
- Cross-platform support (macOS, Windows, Linux)
- Electron-based desktop application framework
- Responsive design with clean button layout
- Basic error handling and input validation

### Technical Features
- Built with Electron v27.3.11
- HTML5, CSS3, and vanilla JavaScript
- Custom CSS styling with dark theme
- Proper window management and menu system
- Build system with electron-builder
- Support for multiple installer formats (.dmg, .exe, .deb, etc.)
