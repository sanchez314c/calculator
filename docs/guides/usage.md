## üÜò Troubleshooting

### Common Issues

**Calculator won't respond to keyboard:**
- Click inside the calculator window to focus
- Check if other applications are capturing keyboard input

**Display shows unexpected results:**
- Verify input sequence matches intended calculation
- Use `Clear` to reset and start over

**Interface appears too small/large:**
- Resize the window by dragging edges
- Check system display scaling settings
